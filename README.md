# Insurance-Data-Interchange-Platform
An insurance claim fraud detection system.
This prototype was designed by Tatenda Alistair Gonzo a young Zimbabwean Tech enthusiast, the system makes use of PHP, Xampp, Javascript and CSS. 
 Insurance Data Interchange Platform allows users to login, enter insurance claims and check for duplicate claims already in the system. 
To install this system onto your machine you have to install XAMPP, and activate Apache and MySql module.
Move the downloaded file and copy it into htdocs.
Open phpmyadmin and import database found in the System File.
